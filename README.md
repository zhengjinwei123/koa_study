# koa_study
学习koa框架

## koa 很多语法基于ECMAScript6,首先需要让编辑器支持ES6

`打开webstorm->File->Settings->Languages & FrameWorkes->勾选ES6即可`

## 运行

`node --harmony app.js`

## 参考资料

ES6 参考资料[阮一峰ES6入门](http://es6.ruanyifeng.com/)
